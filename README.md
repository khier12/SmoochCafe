# SMOOCH CAFE ONLINE ORDERING SYSTEM
### College of Computer Studies – Trimex Colleges
### WEBDEV 1 Final Project

---

## Project Overview
Smooch Cafe Online Ordering System is a web-based project developed to provide an online platform for customers to view and order menu items from Smooch Cafe. The website presents food, drinks, and pastries in a clean and organized layout. It focuses on usability, responsive design, and ease of navigation. This project demonstrates the application of basic web development technologies in creating a functional and visually appealing website.

---

## Project Objective
The objective of this project is to develop a complete and user-friendly website for Smooch Cafe. The system allows customers to browse menu items online and view product details easily. It aims to enhance customer experience through smooth navigation and responsive design. The project also simulates an online ordering process using a shopping cart and checkout page.

---

## Website Pages
- **Home Page** – Displays cafe branding, featured products, and navigation
- **Products Page** – Shows complete menu with category filtering
- **Checkout Page** – Collects customer order and contact details

## Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling, layout, and animations  
- **JavaScript** – Cart functionality, filtering, and interactivity  
- **Font Awesome** – Icons  
- **Google Fonts** – Typography  

---

## Technical Notes
- The shopping cart uses browser localStorage to temporarily save selected items.
- The login feature is for interface demonstration purposes only.
- The website is fully responsive and compatible with desktop and mobile devices.
- This project does not include a backend server or database.
